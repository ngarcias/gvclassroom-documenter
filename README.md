# GVClassroom Documenter

Aplicación full-stack (React + Express) para gestión y documentación de clases, alumnos y asistencia.

## Stack
- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Auth: Passport Local + Sessions
- ## Base de datos

Este proyecto usa SQLite en modo archivo local para facilitar demos y pruebas.

El archivo se crea automáticamente en: ./dev.db



## Requisitos
- Node.js 18+ (recomendado 20)
- (Si aplica) PostgreSQL

## Instalación

1. Clonar el repositorio
2. Crear archivo `.env` desde `.env.example`
3. Ejecutar:

npm install  
npm run dev  

4. Abrir en el navegador:

http://localhost:5000
